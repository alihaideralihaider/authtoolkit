# Product Parts Library

This folder is a staging area for reusable agent and automation ideas before they become real AuthToolkit products.

Product parts are not production features. They are lightweight notes, patterns, and scaffolds used to evaluate whether a workflow should become:

- an AuthToolkit API product
- a SaanaOS feature
- a Kepler Services feature
- a standalone agent or automation package

Use this area to collect repeatable patterns, compare product potential, and document the minimum viable version before implementation. Do not treat anything here as committed product behavior until it has a clear owner, API contract, data model, and compliance review.
